### Prerequisites

- Node.js
- MongoDB

### Installation

```bash
# Clone the repo
git clone https://github.com/Shivanshu-Jha/PingUp.git
cd PingUp
```

# Install server dependencies
cd server
npm install

# Install client dependencies
cd ../client
npm install

Running Locally

# Start backend
cd server
npm start

# Start frontend
cd ../client
npm start


The project is deployed using Vercel. To deploy your own version:
vercel deploy
